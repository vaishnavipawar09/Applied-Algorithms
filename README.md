**Applied Algorithms - Fall 2023**

📌 Course Overview

This repository contains solutions from the Applied Algorithms course (CSCI-B505/INFO-I500) in Fall 2023. The coursework focused on algorithm design, analysis, and problem-solving using Python, covering topics such as complexity analysis, sorting, graph algorithms, dynamic programming, and more.

🏆 Key Topics Covered
- Algorithm Analysis & Asymptotic Notation
- Data Structures (Arrays, Linked Lists, Trees, Queues, Stacks)
- Divide and Conquer Algorithms
- Dynamic Programming
- Greedy Algorithms
- Graph Algorithms (BFS, DFS, Shortest Path)
- Hashing Techniques & Randomized Algorithms
- Streaming Algorithms

📚 References
- The Algorithm Design Manual - Steven Skiena
- Introduction to Algorithms - Cormen et al.
- Data Structures and Algorithms in Python - Goodrich et al.
